156  // Description:
157  // Optionally, set a mask input. This mask may be a binary mask or a label
158  // map. This must be specified via SetMaskType.
159  //
160  // If the mask is a binary mask, the volume rendering is confined to regions
161  // within the binary mask. The binary mask is assumed to have a datatype of
162  // UCHAR and values of 255 (inside) and 0 (outside).
163  //
164  // The mask may also be a label map. The label map is allowed to contain only
165  // 3 labels (values of 0, 1 and 2) and must have a datatype of UCHAR. In voxels
166  // with label value of 0, the color transfer function supplied by component
167  // 0 is used.
168  // In voxels with label value of 1, the color transfer function supplied by
169  // component 1 is used and blended with the transfer function supplied by
170  // component 0, with the blending weight being determined by
171  // MaskBlendFactor.
172  // In voxels with a label value of 2, the color transfer function supplied
173  // by component 2 is used and blended with the transfer function supplied by
174  // component 0, with the blending weight being determined by
175  // MaskBlendFactor.
